# excel
This repository showcases a collection of interactive Microsoft Excel dashboards designed to analyze and visualize real-world business data. The projects demonstrate strong skills in data cleaning, transformation, analysis, and visualization, with a focus on delivering actionable insights for decision-making.


**Tools & Skills**
Microsoft Excel (Advanced)
Data Cleaning & Transformation
Data Visualization & Dashboard Design
Pivot Tables & Charts
Lookup Functions (XLOOKUP, VLOOKUP, INDEX-MATCH)


**Objective**
The goal of this project is to demonstrate the ability to turn raw data into meaningful insights through well-structured dashboards and analysis, helping businesses make informed decisions.
